Flow entry filter on Record_Trigger_Opportunity_Shopify_Automations excludes Shopify Network orders (Transaction_Source__c value "Affiliate" not in DTC filter "POS" OR "Store").

## Decision

Escalated to Engineering. Filter logic requires configuration change to include "Affiliate" source code in addition to existing "POS" and "Store" values.

## Actions

Engineering owns: Update flow entry filter condition to include Transaction_Source__c = "Affiliate" or modify filter logic to accept all legitimate Shopify transaction sources.
