### Technical Constraint

Salesforce Report Builder configurations are stored in the Report metadata but are **NOT directly modifiable via Metadata API or REST API**. Column additions require interactive Report Builder UI:

1. Open Report in Lightning Report Builder
2. Click "Add Column" button
3. Select field: Account_Billing_State__c
4. Click "Add Column" button
5. Select field: Account_Main_Practitioner__c
6. Save report

**Direct API Approach:** Salesforce reports use a proprietary binary/XML format in the metadata API. Attempts to retrieve via `sf project retrieve start --metadata Report:...` return empty XML nodes. The Report Builder UI remains the authoritative interface for column configuration.
