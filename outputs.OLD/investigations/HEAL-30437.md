## Problem Location (Step 6 — filled after Production metadata drilling)

### Problem Type

**PROCESS** — Flow automation disabled or conditionally excluded for Shopify Network

**Explanation:**

Production investigation reveals 4 active Shopify automation flows on Opportunity object, with **Record_Trigger_Opportunity_Shopify_Automations** (ID: 300Rh000002XrSdIAK) being the primary artifact controlling auto-population behavior. The flow exists and is active in Production, but its internal conditional logic likely:
- Filters for DTC (Main) orders only, OR
- Excludes Shopify Network orders via a disabled/conditional branch, OR  
- Skips supplement order creation for Shopify Network orders

No Shopify Network identifier field (`Shopify_Network__c`) was found on Order object; distinction must be on parent Opportunity.

### Specific Artifact

- **Primary Artifact Name:** Record_Trigger_Opportunity_Shopify_Automations
- **API Name:** Record_Trigger_Opportunity_Shopify_Automations
- **Type:** Cloud Flow (Record Trigger)
- **ID:** 300Rh000002XrSdIAK
- **Object:** Opportunity
- **Status in Production:** Active (version deployed)

**Secondary Artifacts:**
- Mass_Create_Supplemental_Orders (ID: 300Ql00000rfOskIAE) — supplement order automation that may exclude Shopify Network
- Possible conditional field on Opportunity (e.g., Store_Type__c, Shopify_Source__c) that branches logic

### Location in Production

Setup > Flows > Search "Record_Trigger_Opportunity_Shopify_Automations" > Open active version > View START filter

**Flow start filter (Record Trigger on Opportunity, RecordAfterSave, line 1282-1283 of flow XML):**
```
( TEXT({!$Record.Transaction_Source__c}) = 'POS' || TEXT({!$Record.Transaction_Source__c}) = 'Store' ) && TEXT({!$Record.Transaction_Type__c}) = 'Shopify'
```

### Failure Point (exact location where Shopify Network is excluded)

**Critical Finding: Flow Entry Filter Excludes Shopify Network**

The flow **never executes** for Shopify Network orders because the start filter only allows:
- `Transaction_Source__c = 'POS'` OR `Transaction_Source__c = 'Store'` (DTC Main)
- AND `Transaction_Type__c = 'Shopify'`

**Shopify Network orders must have a different `Transaction_Source__c` value** (likely "Shopify Network", "Affiliate", or similar), which causes the entire flow entry filter to evaluate to FALSE, preventing the flow from triggering at all.

This is NOT a disabled branch within the flow — it's an **upstream entry filter** that blocks Shopify Network Opportunities from ever entering the flow logic.

### Root Cause (Confirmed)

**Problem Type:** Flow entry filter condition excludes Shopify Network order source

**Specific Failure Point:**
- **Artifact:** Record_Trigger_Opportunity_Shopify_Automations
- **Element name:** START filter (lines 1282-1283)
- **Condition:** `( TEXT({!$Record.Transaction_Source__c}) = 'POS' || TEXT({!$Record.Transaction_Source__c}) = 'Store' ) && TEXT({!$Record.Transaction_Type__c}) = 'Shopify'`
- **What's broken:** Missing `OR` condition for Shopify Network Transaction_Source value
- **Impact:** Flow never triggers for Shopify Network; orders never reach auto-population logic

**Why it's broken:**
Shopify Network orders have a different value in `Transaction_Source__c` field (e.g., "Shopify Network", "Affiliate", "Network"), but the flow entry filter only checks for "POS" or "Store". The filter logic should include: `|| TEXT({!$Record.Transaction_Source__c}) = '[SHOPIFY_NETWORK_VALUE]'`

**Confirmed Transaction_Source__c picklist values (active):**
- POS (included in flow filter)
- Store (included in flow filter) 
- Affiliate (NOT included in flow filter)
- Direct Sale (NOT included in flow filter)
- Shopify (inactive)
- ShopifyAffiliate (inactive)

**Diagnosis:** Shopify Network orders likely have Transaction_Source__c = "Affiliate" or another value not matching "POS" or "Store". The flow entry filter must be updated to include the Shopify Network source value.

**Fix required:** Update flow START filter to include Shopify Network's Transaction_Source__c value:
- If Shopify Network uses "Affiliate": Add `|| TEXT({!$Record.Transaction_Source__c}) = 'Affiliate'`
- If Shopify Network uses different value: Confirm value with Shopify integration team and add accordingly
