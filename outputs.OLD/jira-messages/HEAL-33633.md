## [INTERNAL]

**Root Cause:** CMT JSON path configuration mismatch. The `API_Parameter__mdt` records for Wellvi shipping address fields were mapping values to nested lowercase JSON paths (e.g., `$.shippingAddress.city`), but the Wellvi/TenX API contract expects root-level PascalCase fields (e.g., `$.City`).

**Fix Applied:** Updated four CMT records in Sandbox to use correct JSON paths:
- `API_Parameter.Wellvi_SP_Shipping_City` → `$.City`
- `API_Parameter.Wellvi_SP_Shipping_Street` → `$.AddressLine1`
- `API_Parameter.Wellvi_SP_Shipping_State` → `$.StateProvince`
- `API_Parameter.Wellvi_SP_Shipping_Zip_Code` → `$.PostalCode`
