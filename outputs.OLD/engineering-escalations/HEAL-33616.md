# Engineering Handoff

## Engineering Message

**Issue:** HEAL-33616 — "Update Lab Order" feature throws an error on Lab_Order__c records after height/weight field update
**Problem:** When a support agent uses the "Update Order" feature on a Lab Order record in Salesforce (after editing height/weight fields), an error is returned and the order does not process. This has occurred on at least two separate Lab Order records. The reporter has intentionally not re-attempted submission to preserve the error state for investigation.
**Potential Fix:** Identify the Apex trigger, Flow, or outbound API callout executing on the Lab_Order__c update path that is producing the error. Determine whether the failure is due to a logic/validation condition triggered by height/weight values, a required field missing from the update payload, or an external lab vendor API rejection. Apply targeted fix in Sandbox and validate before deploying to Production.
