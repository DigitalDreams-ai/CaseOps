# Engineering Handoff — HEAL-30437

## Engineering Message

**Issue:** HEAL-30437 — Automate Shopify Network Order Release Process

**Problem:** 
The record trigger flow controlling order auto-population for Shopify orders has an entry filter that only matches DTC (Main) Shopify transactions. Shopify Network orders (which have a different Transaction Source value) are excluded by the filter and never trigger the flow, preventing auto-population and forcing manual order releases.

**Potential Fix:** 
Update the flow entry filter condition on Record_Trigger_Opportunity_Shopify_Automations to include the Shopify Network transaction source value (likely "Affiliate") alongside the existing DTC filter values "POS" and "Store".

---

## Steps to Reproduce

**Affected record IDs:** Any Shopify Network opportunity created in Production

**Steps:**
1. Log into Salesforce Production as a Supply Chain user

3. Create or observe an existing Shopify Network opportunity (Transaction_Source__c = "Affiliate")
4. Observe: Order is created but NOT auto-populated/released

**Expected behavior:** 
Shopify Network orders auto-populate/release matching DTC (Main) behavior, eliminating manual release process

## Root Cause

The record trigger flow `Record_Trigger_Opportunity_Shopify_Automations` (ID: 300Rh000002XrSdIAK) contains an entry filter that explicitly checks for DTC (Main) Shopify sources only. The filter condition is:

```
( Transaction_Source__c = 'POS' OR Transaction_Source__c = 'Store' ) AND Transaction_Type__c = 'Shopify'
```

Shopify Network opportunities have `Transaction_Source__c = 'Affiliate'`, which does not match "POS" or "Store". The flow entry condition evaluates to FALSE, flow never triggers, and auto-population never occurs.

## Potential Fix

**Option 1 (Recommended):**
Update the flow entry filter to include Shopify Network:
```
( Transaction_Source__c = 'POS' OR Transaction_Source__c = 'Store' OR Transaction_Source__c = 'Affiliate' ) 
  AND Transaction_Type__c = 'Shopify'
```

**Option 2 (Alternative):**
If Affiliate handling should differ from DTC, create a parallel flow path for Affiliate orders that mirrors DTC auto-population logic.

**Testing:** 
- Verify DTC (Main) auto-population unaffected after change
- Verify Shopify Network orders now auto-populate
- Confirm supplement order creation includes Shopify Network
