# Engineering Escalation - HEAL-33562

## Issue Summary

Tyler Kaplan (Clinical/RN) cannot recall approved Opportunities. Recall button missing from Approval History dropdown. Order Processing team can recall theirs. Issue affects all Nurses on Shopify and Superhuman Protocol opportunities submitted via automation.

## Root Cause

Two flows auto-submit Opportunities for approval WITHOUT specifying the `submittedBy` parameter:
- Record_Trigger_Opportunity_Shopify_Automations
- Record_Trigger_Opportunity_Superhuman_Protocol_Submit_For_Approval

Salesforce defaults submitter to system/automation user when `submittedBy` is not specified. Salesforce Approval Process only allows the original SUBMITTER to recall. Result: Record owner cannot see Recall button.

## Steps to Reproduce

1. Log in as Tyler Kaplan (Clinical profile, Registered_Nurse permission set)
2. Navigate to Opportunity `006Ql00000eMscDIAS` (or create new Shopify/Superhuman Protocol Opportunity in Open stage)
3. Once opportunity auto-submitted for approval (stage changes to "Waiting for Approval")
4. Scroll to Approval History related list
5. Click dropdown arrow on approval record
6. Observe: Recall option is missing/hidden

**Expected behavior:** Recall option visible for owner to use

**Actual behavior:** Recall option hidden—button only visible to system user (automation), not owner

## Proposed Solution

Add `submittedBy=$Record.OwnerId` input parameter to submit actions in both flows:

```xml
<inputParameters>
    <name>submittedBy</name>
    <value>
        <elementReference>$Record.OwnerId</elementReference>
    </value>
</inputParameters>
```

**Flows to modify:**
1. Record_Trigger_Opportunity_Shopify_Automations.flow-meta.xml
   - Action: Submit_for_Approval
   - Location: Lines 622–627 region (after comment parameter)

2. Record_Trigger_Opportunity_Superhuman_Protocol_Submit_For_Approval.flow-meta.xml
   - Action: Send_to_Approval
   - Location: Lines 11–15 region (after objectId parameter)

**Why this works:** Designates record owner as submitter, allowing them to recall their own approvals per Salesforce native Approval Process behavior.

**Impact:** Restores recall capability for all Nurses/Clinical users on Shopify and Superhuman Protocol opportunities.
