Setup Claude Code

Save the OAuth token used by the CaseOps pipeline

1Generate a token
On your local machine, run:
claude setup-token
Complete the browser authorization if prompted. The command prints a long-lived OAuth token to the terminal.
2Copy the token
Copy only the token printed by claude setup-token.
If you paste CLAUDE_CODE_OAUTH_TOKEN=... or export CLAUDE_CODE_OAUTH_TOKEN=..., CaseOps will save only the token value. Do not paste shell prompts, logs, or ~/.claude/.credentials.json contents.
3Paste below
CaseOps stores this as CLAUDE_CODE_OAUTH_TOKEN in the active env file. On NAS, that is the stack .env file mounted into the container.
