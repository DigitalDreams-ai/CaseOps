Claude Code CLI Authentication

Setup Claude Code authentication in 3 steps:

Run command locally, paste token. Works on mobile & desktop.

Steps:
1. Open the Setup window above (works on mobile)
2. On your local machine, run: claude setup-token
3. Copy the JSON output and paste into the setup window

⚠️ Token Expired?
Run claude logout && claude login locally, then repeat steps above.
